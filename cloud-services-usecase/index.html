Infrastructure & Cloud Solution Use Case

Private cloud for The Largest Telco Company in Indonesia


Customer Challenges

  • On-time application releases
  • Various IT infrastructure and system as deployment target need to be covered in delivering CI/CD automation.


Solution

  • Develop containerized Pre-Production environment as CI/CD deployment target with Kubernetes as the orchestrator:
  • Kubernetes Implementation Services
  • Kubernetes Manage Operation


Result

  • Container-based private cloud, the pre-production environment can run with minimal IT infrastructure, which can be flexibly scaled up when needed (automatic scale)
  • Through this approach, the pre-production environment has become a cost-effective solution in achieving customer satisfaction
